## TLink-sms

# Tool
- This tool send unlimited messages in any India phone number

## Available On
- Termux
- Kali Linux
- Parrot Linux
- Red Hat Linux

## Tested On
- Termux 
- Kali Linux
- Parrot Linux
- Red Hat Linux

## Setup To Run File on Linux
- chmod +x *
- ./Linux-Setup.sh

## Installation 
- apt update && apt upgrade
- apt install git
- git clone https://github.com/KaliLinux037/TLink-sms
- cd  TLink-sms
- chmod +x *
- ./TLink-sms.sh
